## Setup Instructions
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/fravilarc12/TFG.git
   cd TFG
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Prepare the data:
   - Place your raw data files in the `data/raw` directory.
   - Run the data loader script to preprocess the data.

## Usage Guidelines
To run the main application, execute the following command:
```
python src/main.py
```

For exploratory data analysis, open the Jupyter notebook located in the `notebooks` directory:
```
jupyter notebook notebooks/01-exploration.ipynb
```
