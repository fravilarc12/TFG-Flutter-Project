# Results Documentation

This directory contains the results obtained from the experiments conducted as part of the TFG project. 

## Overview

The results are organized based on the different experiments performed, and each result file includes detailed information about the experiment setup, metrics, and findings.

## Usage

To view the results, navigate to the respective experiment folders and open the result files. Each file will provide insights into the performance of the models and methodologies applied during the project.

## Contribution

If you wish to contribute to the results section, please ensure that your results are reproducible and well-documented. Follow the project's guidelines for consistency.
